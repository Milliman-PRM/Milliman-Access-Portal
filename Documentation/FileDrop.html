Permissions within a File Drop

expand_more

File Drop permissions are managed by File Drop Admins. The three types of permissions available are download, upload, and delete, and permissions may vary between users and/or File Drops.

  • Download: The ability to download files from the File Drop
  • Upload: The ability to create folders and upload files to the File Drop
  • Delete: The ability to delete folders and files from the File Drop

Managing Folders and Files within a File Drop

expand_more

Please note that the ability to perform the actions listed below is based upon your assigned permissions for the File Drop. Please see above for definitions of the available permission types.

Navigating the Folder structure

To move into a folder, click the folder name. To navigate back, click the folder you would like to navigate back to at the top of the Files box.

Adding a Folder

  1. Navigate to the location that you would like to add a folder
  2. Click on "Add Folder"
  3. Enter the name of the folder that you would like to create
  4. Enter a description (optional)
  5. Press the checkmark to save or the "x" to cancel

Adding a File

There are a few ways to add a file. You can drag and drop a file into the section, or you can click "Add File". See below steps for adding a file.

  1. Navigate to the location that you would like to add a file
  2. Click on "Add File"
  3. Choose file to upload

You can cancel the upload of the file by pressing the "x" while it is uploading.

Note that you can add multiple files at a time. You can see the files that are currently being uploaded in a single place by expanding the banner on the File Drop card.

Editing a Folder

  1. Click on the vertical ellipses to the right of the folder to edit
  2. Click on "Edit"
  3. Edit the folder name and/or the description
  4. To save, press the checkmark. To cancel, press the "x".

Moving a File or Folder

  1. Click on the vertical ellipses to the right of the file or folder to move
  2. Click "Move"
  3. There are several options that can be taken at this point. See below for various ways to move the file or folder:
    • Navigating up the folder structure: Use the breadcrumb links to navigate up the folder structure. Once you have navigated to the folder where you want to move the folder or file, press "Move"
    • Navigating down the folder structure: Click the folder listed in the modal to enter that folder and then press "Move"
    • Create a new folder: Press "Create new folder" to create a folder within the folder you are currently in, enter the folder name and save, and then you can move the folder or file into the new folder by pressing "Move"

Deleting a File or Folder

  1. Click on the vertical ellipses under to the right of the file or folder to delete
  2. Click "Delete"
  3. Press "Delete" on the popup notification

Downloading a File

  1. Navigate to the file to download
  2. Click on the filename to download

Interacting with a File Drop via SFTP

MAP File Drop features for uploading, downloading, and deleting folders and files are supported using Secure File Transfer Protocol (SFTP). Use of an SFTP client application such as FileZilla is the means to interact with the folders and files of a MAP File Drop. You can find information regarding the SFTP settings via the "My Settings" tab.

My Settings

expand_more

Please note that access to the sections listed below is based on permissions currently assigned to you for this File Drop.

SFTP Connection:
This section provides connection details of MAP's SFTP server. Use these details to configure an SFTP Client application such as FileZilla

SFTP Credentials:
This section displays your SFTP login information for the currently displayed file drop. Please note that the username is case sensitive

The password can be reset at any time by clicking the button. The new password will be displayed and you can copy that to the clipboard using the provided button.  This is the only time that the password can be seen so please store it securely, such as in a password manager application

Notification Settings:
You can choose to receive email notifications each time certain file related events occur in the file drop. Use the toggle control to subscribe

Subscription changes apply only to the file drop currently displayed, and are effective immediately
