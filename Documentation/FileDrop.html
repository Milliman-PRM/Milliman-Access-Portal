File Drop User Guide

This user guide is intended to walk through how to manage file drops. Some of the content found in this user guide will pertain to the Client Administration tab within Milliman Access Portal (MAP).


File drop roles are set within a client. This means that the roles are set in the Client Administration tab of MAP. To set the roles for a user of the client, navigate to the Client Administration tab, filter to the client by clicking the client card, and then open the user card in the rightmost column. The user roles contain File drop user and File drop admin.

  • File drop user: A user who is capable of uploading, downloading, and deleting content (these permissions are set within a file drop), and can manage their own settings, but cannot view the file drop activity log or user permissions.
  • File drop admin: A user with the ability to set user permissions at the file drop level. A file drop admin can also view the activity log for the file drop as well as their settings.

Creating a File Drop

  1. Choose the client that you would like to create a file drop for
  2. Click "New File Drop"
  3. Fill in the file drop name and description (optional) and press "Add"

Editing a File Drop

  1. Navigate to the file drop to edit
  2. Click on the blue pencil icon on the file drop card in order to enter edit mode
  3. Click into the file drop name and/or description to edit
  4. Press the green check mark to save

Deleting a File Drop

  1. Navigate to the file drop to delete
  2. Click on the trash icon on the file drop card
  3. Confirm deletion of the file drop

Creating a Permission Group

  1. Navigate to the file drop that you want to add a permission group to
  2. Click the pencil icon in the upper right to add User Permissions
  3. To add a user with their own permissions, click "Add User". To add a group of users with the same permissions, click "Add Group"
  4. Choose the user(s) to be added from the dropdowns and select which permissions they should receive between Download, Upload, and Delete
  5. Press "Save Changes"

Note that only users who have been given the role of "File Drop User" from the Client Administration page can be added to the User Permissions for a file drop. Also note that a user can only be assigned once per file drop. This means that if a user is a part of one permission group, they cannot be added to a different permission group for the same file drop.

Editing a Permission Group

  1. Navigate to the file drop that contains the permission group that you would like to edit
  2. Click on the edit icon in the upper right of the User Permissions form
  • Delete a user/permission group: Pressing the trash icon for a user or permission group will remove permissions for the user/group to the file drop
  • Add users/permission groups: Press "Add User" or "Add Group" to create more permission groups
  • Remove users from a specific permission group: Pressing the minus icon next to a user in a permission group will remove that user from the permission group
  • Edit permission group name: Edit a permission group name by clicking into the name and editing it

Setting Permissions for a Permission Group

  • Download: The ability to download files from the file drop
  • Upload: The ability to upload files to the file drop
  • Delete: The ability to delete files from the file drop

As of now, in order to upload, download, or delete files, you have to use a SFTP client such as FileZilla. The next release of MAP will include an interface for this action.

Managing Files using SFTP

  1. Within MAP, navigate to the My Settings tab
  2. Click "Generate Credentials" ********check this
  3. Open a SFTP client outside of MAP
  4. Navigate to the site manager in the SFTP client
  5. Fill in the SFTP SSH file transfer protocol
  6. Fill in the host, port, and username information
  7. Copy the password generated by MAP and use in the SFTP client
  8. Connect to MAP

The activity log is available so that file drop admins can see what has been taking place within the file drop. You have the ability to refresh the page as needed. 30 days worth of data is shown by default, but pressing the "Download All" button will download all of the data available for the file drop as a csv.
